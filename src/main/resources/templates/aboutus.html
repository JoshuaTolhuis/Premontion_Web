Gene expression experiments do not always give a complete connected protein-to-protein interaction network.
Therefore Dr. J. Bosman Et al. developed Premonition. Premonition takes a set of candidate genes from I.E.
RNA-sequencing, and constructs a as probable as possible protein-to-protein interaction network.
It does this by adding genes from which is known they interact with the candidate genes.
Next the program will remove as much uninformative genes as possible, so a scarce network remains.
The constructed network is now enriched with biological relevant genes and a Gene Ontology (GO) analysis which shows strong enrichment of biological processes.


Premonition produces an interactive 2D protein-to-protein network based on known information about a specific system or organism. This network is downloadable and is usable with Cytoscape powered systems.

Authors: Premonition: J.Bosman, Z. Eelderink-Chen, E.Laing, M. Merrow.
